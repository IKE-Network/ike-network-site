# Maven Coordinates

## [Apache Maven](#apache-maven)

```
<dependency>
  <groupId>network.ike</groupId>
  <artifactId>ike-network-site</artifactId>
  <version>2</version>
  <type>pom</type>
</dependency>
```

## [Apache Ivy](#apache-ivy)

```
<dependency org="network.ike" name="ike-network-site" rev="2">
  <artifact name="ike-network-site" type="pom" />
</dependency>
```

## [Groovy Grape](#groovy-grape)

```
@Grapes(
@Grab(group='network.ike', module='ike-network-site', version='2')
)
```

## [Gradle/Grails](#gradle-grails)

```
implementation 'network.ike:ike-network-site:2'
```

## [Scala SBT](#scala-sbt)

```
libraryDependencies += "network.ike" % "ike-network-site" % "2"
```

## [Leiningen](#leiningen)

```
[network.ike/ike-network-site "2"]
```
