= IKE Network
:icons: font

The IKE Network (Integrated Knowledge Exchange) is a sociotechnical
fabric where knowledge compounds.

== Foundation

The IKE foundation — published to Maven Central and
inheritable by any project.

The foundation members fall into two layered orderings.
`ike-base-parent` sits at the apex of the *parent*
inheritance chain — every other foundation artifact
inherits from it. `ike-platform` sits at the terminus
of the *build-and-release* dependency chain — releases
propagate through it last.

== Examples

Reference projects that show how to consume the IKE
foundation. They are deliberately *not* published to
Maven Central: they are worked examples to read and
copy, not libraries to depend on. Publishing them as
artifacts would invite accidental coupling to code
that exists only to illustrate a pattern.
